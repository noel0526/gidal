## 0913

### 회의록

+ 식당게시판은 가로형식으로 구현

+ 상세페이지는 따로 존재하지 않으며, 예약과 웨이팅 버튼을 통해서 세부 내용이 확인 가능하다.

+ 이벤트 페이지 설계

+ 기업페이지, 일반페이지, 관리자 페이지 어떤식으로 만들것인가 정리

+ 리뷰는 내가 예약하거나 웨이팅한 곳만 달수 있음

+ 리뷰를 할수 있는 위치는 일반페이지의 내 예약목록 및 내 웨이팅 목록에서 리뷰글쓰기 가능

+ 기업도 댓글쓰기는 기업페이지에서 가능하다
